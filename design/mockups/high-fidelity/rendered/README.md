# High-Fidelity Rendered Images

Status: active v1

Dieser Ordner enthaelt die aktuellen gerenderten High-Fidelity-Bilder fuer Vivicast.

## Rolle

Gerenderte PNGs sind visuelle Stilreferenzen.

Sie sind keine normative Quelle fuer:

- Produktumfang
- Navigation
- sichtbare Labels
- UI-Texte
- Fokuspfade
- Datenmodell
- Architektur
- Sicherheitsverhalten
- Backup/Restore-Verhalten
- PIN-Regeln
- Playback-Regeln

Bei Konflikten gelten die aktiven Markdown-Quellen nach `DOCS-GOVERNANCE.md`.

## Aktuelle Renderings

Neue v1-Renderings werden direkt in diesem Ordner abgelegt.

## Archiv

Der Ordner `archive/` enthaelt historische Renderings.

Codex muss `archive/` fuer aktuelle App-Umsetzung ignorieren, ausser der Owner fordert ausdruecklich einen historischen Vergleich an.

Historische Renderings duerfen nicht als aktuelle visuelle Zielquelle verwendet werden.
