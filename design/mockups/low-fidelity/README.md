# Low-Fidelity Mockups

Status: vorhanden v1

Dieses Verzeichnis ist fuer Low-Fidelity Layout Previews vorgesehen.

Low-Fidelity Mockups dienen nur zur Pruefung von Struktur, Fokuspositionen und Screen-Dichte.

Sie sind nicht die finale visuelle Zieloptik von Vivicast.

## Zielstruktur

```text

```

## Zweck

```text
Screen-Aufbau pruefen
Fokuspositionen pruefen
Screen-Dichte grob einschaetzen
Android-TV-Grundlayout validieren
```

## Nicht-Zweck

```text
finales UI-Design zeigen
Premium-Look definieren
pixelgenaue Vorlage sein
```
